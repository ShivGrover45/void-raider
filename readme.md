# 🚀 Void Raider

A 2D arcade-style space shooter built with **Python** and **Pygame CE**.

Control your spaceship, dodge incoming meteors, destroy them with lasers, and survive as long as possible while increasing your score.

---

## 🎮 Features

- 🚀 Smooth player movement
- 🔫 Laser shooting with cooldown
- ☄️ Randomly spawning meteors
- 🔄 Rotating meteors
- 💥 Explosion animations
- 🎯 Real-time score tracking
- 🌠 Animated starfield background
- 🎵 Background music and sound effects
- 🎯 Pixel-perfect collision detection using masks

---

## 🛠️ Tech Stack

- Python 3
- Pygame CE

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/ShivGrover45/void-raider.git
```

Move into the project directory:

```bash
cd void-raider
```

Install the dependencies:

```bash
pip install pygame-ce
```

Run the game:

```bash
python code/game.py
```
